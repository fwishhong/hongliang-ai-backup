# 2026-02-14 记忆

## Seedance 2.0 电影制作工作流

### 核心公式
```
Seedance 2.0 = 多模态参考（@符号）+ 强创意生成 + 精准指令
```

### 角色一致性方法
- 首次：用 Flux/Seedance 生成角色正脸图
- 后续：用 `@角色图` 参考保持一致性
- LoRA 训练：需要 15-20 张图（最优方案）

### 参考符号使用
- `@图片` - 图片参考（角色/场景/道具）
- `@视频` - 运镜/动作参考
- `@音频` - 音效/音乐参考
- 最多：9张图 + 3段视频 + 3段音频

### 完整 Pipeline
1. 角色创建 → Flux 生成角色图
2. 场景创建 → Flux 生成场景图
3. 分镜生成 → 脚本转分镜表
4. 视频生成 → Seedance 2.0 按分镜生成
5. 视频延长 → "接着拍"续接镜头
6. 后期处理 → Foley音效 + 升频 + 合并

### inference.sh 命令
```bash
# 视频生成
infsh app run bytedance/seedance-1-5-pro --input '{
  "image_url": "角色图.jpg",
  "prompt": "角色动作描述"
}'

# 视频延长
infsh app run bytedance/seedance-1-5-pro --input '{
  "video_url": "上段视频.mp4",
  "prompt": "延长：角色继续..."
}'
```
